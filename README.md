# Pseudo-Spectral Motion Planning for UGVs & UAVs

Pseudo-spectral motion planning for UGVs and UAVs with obstacle avoidance. Explores P-norm, normalized P-norm, and soft constraints for clustered obstacles. Extended to point-mass and 3-DOF UAVs, validated using dynamics and differential flatness, with RK45 trajectory simulation.
